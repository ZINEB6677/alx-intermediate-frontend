Responsive Image Gallery

A demonstration of CSS Grid layout with responsive images that adapt to different screen sizes

About This Gallery

This image gallery uses CSS Grid to create a responsive layout. On larger screens, images are displayed in three columns. As the screen size decreases, the layout adjusts to two columns, and finally to a single column on mobile devices.

Responsive Design

All images are fully responsive and will resize properly on different screen sizes. Hover over the images to see a subtle zoom effect and reveal the image captions.
